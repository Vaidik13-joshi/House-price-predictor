# 🏠 House Price Prediction App

A simple and interactive Machine Learning web app that predicts house prices based on user inputs.

---

## 🚀 Live Demo

👉 https://https://house-price-predictor-ywztgb6jwroqxhwdtfkrjd.streamlit.app/

---

## 📌 Features

* Instant house price prediction
* Clean and user-friendly UI
* Built with Machine Learning
* Deployed using Streamlit

---

## 🧠 Tech Stack

Python • Streamlit • Scikit-learn • NumPy • Pandas

---

## 📊 Inputs

Longitude • Latitude • House Age • Total Rooms • Bedrooms • Population • Households • Income

---

## ⚙️ How it Works

User inputs data → Model processes → Price predicted instantly

---

## 📁 Project Structure

House-Price-app/
│── app.py
│── model.pkl
│── requirements.txt
│── README.md

---

## 💻 Run Locally

git clone https://github.com/Vaidik13-joshi/House-price-predictor.git
cd House-price-predictor
pip install -r requirements.txt
streamlit run app.py
